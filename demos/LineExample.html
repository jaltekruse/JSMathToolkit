The Algorithmist::Parametric Line Demo

The demo illustrates the parametric equation of a line in 2D space. Drag the circular Markers to adjust the line endpoints. Drag the vertical GraphSlice tool to query line coordinates and parameter value. The line is drawn using EaselJS with a solid line decorator; the drawing and line decoration is completely decoupled from the computations.

Drag the slice tool (triangular markers) to query parameter and the line coordinates. Drag the circular Markers to change endpoints.
