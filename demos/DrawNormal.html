The Algorithmist::Graph Normal Distribution

The demo illustrates graphing an arbitrary Normal distribution. Adjust the mean and standard deviation. The graph interval is [-4,4] with interpolation nodes. The actual function values (plotted point-to-point) are shown in red and the draw stack computed by the Normal class is shown in white.

Mean     Std. Dev.  

The curve is slightly stretched due to graph aspect ratio, i.e. pixels/unit X != pixels/unit Y. Adjust slider values to control Bezier approximation interval.
