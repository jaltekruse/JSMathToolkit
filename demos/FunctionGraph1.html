The Algorithmist::Function Graph Engine Demo

The demo illustrates the very basic functionality of the function graphing engine. Use the supporting JS to become comfortable with setting up axes, a grid, tic marks, and tic labels.

Zooming is only one level in/out by a factor of 2. Panning is 20px per click.
