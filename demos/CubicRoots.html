The Algorithmist::Cubic Root Finder Demo

This demo exercises the cubic (polynomial) root finder. The root finder works by specifying an interval over which all roots are to be found. If no such interval is known in advance, use a very wide portion of the function's domain. All output is sent to the console, so you will have to view the console for results.
