The Algorithmist::Point On Line Demo

This demo illustrates testing whether or not a point is on a line segment between two other points. Drag the circular Markers to adjust the line-segment points and see the results of the test, which returns true only if the test point is strictly on the line segment and in between endpoints (within numerical tolerance). Some additional information is provided, including the projection of the test point onto the line.
