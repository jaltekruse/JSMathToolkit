The Algorithmist::Function Graph Engine Demo

The demo illustrates how to use the function library in the function graphing engine. This library allows a variety of popular functions to be plotted and each serves as an example that can be used to customize the function library to individual requirements.

Zooming is only one level in/out by a factor of 2. Panning is 20px per click.
