The Algorithmist::Arrow Shape Demo

This demo illustrates interactive creation of vertical or horizontal arrows. Many shapes can be defined by a bounding rectangle and a small number of parameters. This process is illustrated with the arrow by dragging a bounding rectangle and supplying two additional parameters. The polygon vertex coordinates are queried from the Arrow class and the arrow is drawn using EaselJS.

Orientation  

Enter Arrow length as fraction of total (not arrowhead length)

Enter Arrow width as fraction of total (not arrowhead width)

Select the orientation, then arrow (not arrowhead) length as a fraction of the total length. then click and drag a rectangle in the drawing area to create the arrow.
