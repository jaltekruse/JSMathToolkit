The Algorithmist::Line Decorator Demo

The Line class manages the parametric representation of a Line in 2D space. This demo illustrates how to use EaselJS line decorators to render a line. Select the desired line decorator from the drop-down list and drag the circular Markers to adjust the line endpoints. Line rendering is decoupled from line computations in the library.
