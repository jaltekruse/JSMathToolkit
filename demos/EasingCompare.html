The Algorithmist::Basic Easing Demo

Easing maps t in [0,1] to t* in [0,1] in order to alter the motion along a path. This demo illustrates the difference in the two parameter value for a variety of easing methods and default parameter selections for each of the methods. Select easing method and ease in/out or in-out. Then, click start to being a ten-second animation. The eased parameter value is shown in green.

  • Back
  • Bounce
  • Cubic
  • Elastic
  • Exponential
  • Linear
  • Quadratic

Note: All functions are exercised in such a manner that a base parameter, t, is varied from 0 to 1 (red marker) over a time interval of 10 seconds. Colored markers allow quick visual comparison of linear time variance vs. easing.
